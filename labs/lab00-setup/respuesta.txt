lab00 - setup

En este laboratorio usted entrara a un servidor y tambien aprendera
formas de conocer el hardware de la maquina.

Actividad:
1) Acceso a GUANACO (servidor HPC de Informatica)
    a) generar una llave publica ssh (sin passphrase)
        > ssh-keygen -t rsa -b 4096
    b) Enviar la llave publica a su profesor por slack. Enviar el output de:
        > cat ~/.ssh/id_rsa.pub
    c) crear nombre de usuario. Usar [inicial nombre][apellido], ejemplo 'cnavarro'.
    d) Agregar entry de guanaco en config de ssh:
        i) editar archivo ~/.ssh/config
        ii) agregar entry (respetar indentacion):
            Host guanaco
                HostName 146.83.216.158
                Port 22334
                User USUARIO
                IdentityFile ~/.ssh/id_rsa
        iii) guardar archivo, salir.
    e) Entrar a guanaco
        > ssh guanaco


2) Encontrar caracteristicas hardware de guanaco, averigue:
    a) Cantidad de RAM, tipo (DDR cuanto) y velocidad.
        > Usar htop, y lshw
        > 256 GB RAM, DDR4 2666 MHZ.
    b) Cantidad de Almacenamiento y tipo (HDD?, SSD?).
        > Usar: sudo fdisk -l
        > 7 x 1.8TB (/dev/sdX --> conector SATA)
        > SSD (cat /sys/block/sda/queue/rotational -> 0 (ssd))
    c) Cantidad de nucleos CPUs. y si tiene o no hyper-threading (buscar si no conoce el termino)
        > cat /proc/cpuinfo,  lscpu, lshw
        > 28 cores fisicos, no hyperthreading
    d) Cantidad de CPUs, modelo, # nucleos y cantidad de cache
        > 2, Intel Xeon Gold 5117 2GHz, 14 cores/CPU, cache L3 ~ 40MB
    e) Cantidad de GPUs, modelo, memoria, nucleos.
        > 4 RTX 2080 Ti, 11GB VRAM, 4352 CUDA cores.
    f) Tipo de interfaz de red y velocidad.
        > lspci | grep Ethernet
        > Ethernet, GigaBit (confirmar con internet)
